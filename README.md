# 使用说明

1. 下载（[下载页面](#)） 
2. 解压 
3. 切换至文件夹 
4. 确保和可执行文件的同一文件夹下有info-course.txt和info-student.txt两个文件 
5. 执行CurriculaVariable或CurriculaVariable.exe
