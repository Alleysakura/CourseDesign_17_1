# 学生选课系统-CurriculaVariable

--- 

本选课程序使用c语言构建，共计730行代码，拥有完整的注释，变量名可读性强，具有以下特色：

* 1 多平台支持。
* 2 友好的交互界面(命令行环境)

# 使用说明

1. 下载。
2. 解压。
3. 切换至文件夹。
4. 确保和可执行文件的同一文件夹下有info-course.txt和info-student.txt两个文件。
5. 执行CurriculaVariable或CurriculaVariable.exe。

PS: 教师初始管理密码为: 12345678 (抱歉设置了弱密码Orz)。
